Pokemon Brick Bronze Roblox Game Archive.


Usage

Download the .rbxl file from one of the links below.
Then download and open Roblox Studio and upload it there.
Publish it as a private game because Roblox might take it down if it's a public game.
Note: all of the links below are the same. If a link doesn't work, just use another one.
